❓ Como usar o Xestor de BBDD

1. Primeiro uso

Fai clic en "Crear novo ficheiro" e garda un .json nun lugar seguro (ex: Escritorio). Tes os teus datos fóra do navegador desde ese momento.

2. Gardar datos

Fai clic en "Gardar ficheiro" ou activa o autoguardado. Cada vez que traballas coas ferramentas os datos van ao localStorage; este xestor cópiaos ao disco.

3. Recuperar tras limpeza de caché

Abre este xestor → "Abrir ficheiro existente" → selecciona o teu .json → "Restaurar a localStorage". Listo.

4. Firefox

Usa "Copia datada" para descargar e "Importar JSON" para recuperar. O autoguardado non está dispoñible nese navegador.

💡 Consello extra: Garda tamén unha copia nunha memoria USB ou Google Drive.

📥 Importar ficheiro JSON
Selecciona un .json exportado previamente para cargar os datos no localStorage.
